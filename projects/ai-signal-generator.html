Overview & Results

Created a machine learning-based trading signal system that analyzes 50+ technical features in real-time and generates buy/sell signals for forex markets. The system includes a Streamlit dashboard for monitoring performance and receiving alerts via Telegram.

Key Achievement: Achieved 68% directional accuracy on out-of-sample data with Sharpe ratio of 1.8 and 12% maximum drawdown through ensemble learning and rigorous feature engineering.

How It Works

1. Feature Engineering

Engineered 50+ features combining technical indicators (RSI, MACD, Bollinger Bands), price action patterns, volatility metrics, and market microstructure signals. Features are standardized and selected using correlation analysis.

2. Ensemble ML Model

Built an ensemble combining XGBoost, Random Forest, and Logistic Regression. Each model votes on trade direction, with final signal requiring 2/3 consensus. This reduces false signals and improves reliability.

3. Real-Time Dashboard

Deployed Streamlit web app that displays current signals, performance metrics, equity curves, and win/loss statistics. Integrated Telegram bot sends instant alerts when new signals are generated.

Technology & Key Learnings

Tech Stack:

Python 3.9+ XGBoost Streamlit pandas Telegram API

What Worked:

  • Ensemble approach reduced overfitting compared to single models
  • Feature importance analysis revealed top 15 features drove 80% of performance
  • Real-time monitoring caught model drift early

Challenges Solved:

  • Handled missing data with forward-fill and interpolation
  • Optimized refresh rate to balance latency vs computational cost
  • Implemented model retraining pipeline to adapt to regime changes
