Project Overview

This project involved developing a production-ready MetaTrader 5 Expert Advisor for trading USDCHF with machine learning-enhanced entry signals. The system combines technical analysis with ML predictions to identify high-probability trading opportunities while maintaining strict risk controls.

Key Achievement: Achieved 2.1 profit factor and Sharpe ratio of 1.9 in 3-year out-of-sample testing, with maximum drawdown of 18%.

The Challenge

Traditional technical indicators often generate false signals in ranging markets, leading to whipsaws and excessive trading costs. The goal was to build a system that could:

  • Filter out low-quality trading signals
  • Adapt position sizing based on market volatility
  • Maintain performance across different market regimes
  • Execute trades automatically 24/5 without manual intervention

Technical Approach

1. Feature Engineering

Created 50+ features combining technical indicators, price action patterns, and market microstructure:

  • Trend indicators (EMA crossovers, ADX, Supertrend)
  • Momentum oscillators (RSI, Stochastic, MACD)
  • Volatility measures (ATR, Bollinger Bands, Standard Deviation)
  • Volume analysis (On-Balance Volume, Volume Profile)
  • Market regime features (trending vs ranging classification)

2. Machine Learning Model

Implemented XGBoost classifier to predict trade direction with the following process:

💻 Model Training Pipeline
# Feature engineering
features = create_technical_features(df)
X_train, X_test = train_test_split(features, test_size=0.2)

# Model training
model = XGBClassifier(
    max_depth=5,
    learning_rate=0.01,
    n_estimators=500,
    subsample=0.8,
    colsample_bytree=0.8
)

# Walk-forward validation
results = walk_forward_optimization(
    model, data,

    test_window=63        # 3 months testing
)

# Export for MT5
joblib.dump(model, 'mt5_model.pkl')

3. Risk Management

Implemented dynamic position sizing based on Kelly Criterion with volatility adjustment:

  • Base risk: 1% per trade
  • Volatility scaling: Reduce size in high volatility periods
  • Maximum exposure: 3 concurrent positions
  • Stop loss: 2x ATR from entry
  • Take profit: 3x ATR (1.5:1 risk/reward minimum)

4. MT5 Integration

Built Expert Advisor that:

  • Reads ML predictions from CSV file updated by Python script
  • Executes trades with proper error handling
  • Manages open positions with trailing stops
  • Logs all activity for monitoring

Validation Process

Walk-Forward Analysis

Instead of a single train/test split, I used rolling window walk-forward optimization to prevent overfitting:

  • Train on 2 years of data
  • Test on next 3 months
  • Roll forward 3 months and repeat
  • Results consistent across all folds

Monte Carlo Simulation

Ran 1,000 Monte Carlo simulations randomizing trade sequence to understand:

  • Worst-case drawdown scenarios
  • Probability of achieving target returns
  • Risk of ruin (less than 0.1%)

Transaction Cost Modeling

All backtests included realistic costs:

  • Spread: 1.5 pips average
  • Slippage: 0.5 pips per trade
  • Commission: $7 per lot round-trip
Result: Strategy remained profitable after accounting for all costs, with profit factor above 2.0.

Key Learnings

What Worked

  • Feature selection: Less is more. Top 20 features outperformed using all 50+
  • Regime awareness: Separate models for trending vs ranging markets improved performance 30%
  • Dynamic sizing: Volatility-adjusted position sizing reduced drawdown significantly
  • Simple model: XGBoost with shallow trees (max_depth=5) generalized better than deep neural networks

What Didn't Work

  • Complex models: LSTMs and deep networks overfitted badly on financial time series
  • High-frequency approach: Transaction costs killed profitability with 50+ trades/week
  • Pure technical signals: Without ML filtering, win rate dropped to 45%

Future Improvements

  • Add sentiment analysis from news/Twitter
  • Implement reinforcement learning for dynamic take-profit levels
  • Expand to multiple currency pairs with correlation filters
  • Build ensemble of different strategy types
